    mysqlnotlike不能将数据完全查出来

    今天在查数据库的时候执行了一条sql 如下

    #查询语句2

#找出表中id中不含有‘A'且不含有‘B'且不含有‘C'的字段

select * from table1 where name not like '%A%'  ;

#成功查出400条

    正常来说是有500条数据，但是不知为何只有400条，因为是没有把表中 name 为 null的查询出来，所以怀疑是 not like 不能查询这种方式对 null 无效。
